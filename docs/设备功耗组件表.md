# P_device 细化：组件、状态、典型功耗与备注

外电路传入电芯的功率 P_device [W] 由以下组件按状态查表汇总。  
典型功耗取区间中值，用于仿真；如需保守/激进可改用 min/max。

| 组件 (Component) | 状态 (State) | 典型功耗 (Watts) | 备注/引用 |
|------------------|--------------|------------------|-----------|
| 屏幕 (Display) | Off | 0.0 W | OLED 黑色不发光 |
| | Low Brightness | 0.5 W | 室内阅读亮度 (0.4–0.6 W) |
| | High Brightness | 1.35 W | 户外阳光下或 HDR 视频 (1.2–1.5 W) |
| 处理器 (CPU/SoC) | Idle (待机) | 0.075 W | 深度睡眠 Doze Mode (0.05–0.1 W) |
| | Medium Load | 1.0 W | 刷网页、普通 APP (0.8–1.2 W) |
| | High Load | 3.25 W | 3D 游戏 CPU+GPU 满载 (2.5–4.0 W) |
| 网络 (Network) | WiFi Idle | 0.02 W | 保持连接 |
| | WiFi Active | 0.4 W | 视频流下载 (0.3–0.5 W) |
| | 4G/5G Active | 1.75 W | 信号弱时功耗会翻倍 (1.0–2.5 W) |
| GPS | Off | 0.0 W | 关闭 |
| | Tracking | 0.45 W | 持续导航模式 (0.4–0.5 W) |
| 其他 (Background) | Sensors/Audio | 0.15 W | 蓝牙、音频解码等 (0.1–0.2 W) |

- **计算**：`P_device = 屏幕 + 处理器 + 网络 + GPS + 其他`，由 `scripts/device_power_components.py` 中的 `P_device_from_state(display_state, cpu_state, network_state, gps_state, other_state)` 汇总。
- **测试**：运行 `python scripts/test_device_power.py` 或 `python scripts/device_power_components.py` 可打印表并校验若干组合。
